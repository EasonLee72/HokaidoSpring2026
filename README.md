## 📍 行程概覽

| 天數 | 行程 | 主要景點 |
|------|------|----------|
| Day 1 | 札幌半日遊 | 新千歲機場、螃蟹家、狸小路商店街 |
| Day 2 | 札幌市區一日遊 | 二條市場、札幌電視塔、大通公園、時計台、藻岩山 |
| Day 3 | 郊區一日遊 | 圓山公園、北海道神宮、白色戀人公園 |
| Day 4 | 小樽一日遊 | 三角市場、小樽運河、音樂盒堂、堺町通商店街 |
| Day 5 | 登別一日遊 | 尼克斯海洋公園、地獄谷、登別溫泉 |
| Day 6 | 登別、洞爺湖 | 大湯沼川足湯、洞爺湖汽船、溫泉街 |
| Day 7 | 洞爺湖、函館 | 浮見堂、SAIRO 展望台、函館朝市 |
| Day 8 | 函館一日遊 | 五稜郭、元町地區、函館山夜景 |
| Day 9 | 札幌購物 | 狸小路商店街、新千歲機場 |
| Day 10 | 返程 | 溫暖歸途 |

## 📝 使用說明

1. 從 `index.html` 開始瀏覽
2. 點擊「開始旅程」進入目錄頁
3. 選擇想查看的行程項目
4. 使用頁面底部的導航按鈕瀏覽相鄰頁面

## 📷 照片替換

網站中的照片目前使用 emoji 作為佔位符。要替換為真實照片：

1. 將照片放入 `images/` 資料夾
2. 修改 HTML 中的 `.photo-item` 內容：
```html
<div class="photo-item">
  <img src="images/your-photo.jpg" alt="照片描述">
</div>
```
